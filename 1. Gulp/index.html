Gulp

Sposób na automatyzacje pracy

Co to jest?

  • Najlepszy pomocnik :-)
  • Apliacja Node.js
  • Biblioteka

Co pomocnik zrobi za Ciebie?

  • Zarządzania stylami
    • kompilacja z SCSS
    • prefixowanie
    • konkatenacja
    • minifikacja
  • Zarzadzania skryptami
    • kompilacja z Bable/TypeScript
    • minifikacja
  • Automatyczne odświeżanie przeglądarki - LiveReload
  • Minifikacja plików
  • Ułatwia zarządzanie zależnościami np. czcionki, biblioteki zewnętrzne
  • oraz wiele, wiele innych - duży zbiór gotowych wtyczek

Jak obsługiwać?

Wymagany jest Node.js na komputerze.

  1. Zainstalować gulp 
    $ npm install --global gulp-cli
$ npm install --save-dev gulp
  2. Zainstalować wtyczki gulp 
    $ npm install gulp-sass gulp-uglifyjs --save-dev
  3. Stworzyć plik gulpfile.js
    • 6 prostych metod: gulp.task, gulp.src, gulp.dest, gulp.watch, gulp.run, .pipe
  4. Uruchamiać, uruchamiac, uruchamiać, uruchamiać

gulp.task

gulp.task(name, [dep], fn)

Definiuje nowe zadanie o nazwie name. Zadanie okreslane jest przez funkcje fn.

gulp.task('scss', function () {
    return gulp.src('./source/*.xml')
        .pipe(gulp.dest('./destination/'));
});

gulp.src

gulp.src(paths)

"Otwiera" pliki z scieżki paths i zwraca strumień

gulp.task('scss', function () {
    return gulp.src('./source/*.xml')
        .pipe(gulp.dest('./destination/'));
});

gulp.dest

gulp.dest(path)

Zapisuje strumień do katalogu path.

gulp.task('scss', function () {
    return gulp.src('./source/*.xml')
        .pipe(gulp.dest('./destination/'));
});

gulp.watch

gulp.watch(path, tasks);

Zaczyna obserwowac pliki path i uruchamia zadania task.

gulp.task('watch', function () {
    gulp.watch('./assets/scss/*.scss', ['scss']);
});

gulp.run

gulp.run(tasks);

Uruchamia zadania tasks.

gulp.task('compile', function () {
    gulp.run(['scss', 'js', 'images']);
});

.pipe

.pipe(plugin)

Transformuje strumień za pomoca wtyczki plugin.

return gulp.src('./assets/scss/style.css')
    .pipe(autoprefixer())
    .pipe(gulp.dest('./static/css/'));
    .pipe(livereload());
var gulp = require('gulp');
var scss = require('gulp-scss');
var autoprefixer = require('gulp-autoprefixer');
var livereload = require('gulp-livereload');
var uglify = require('gulp-uglify');
var concat = require('gulp-concat');
var rename = require('gulp-rename');

gulp.task('scss', function () {
    return gulp.src('./assets/scss/style.scss')
        .pipe(scss())
        .pipe(autoprefixer())
        .pipe(gulp.dest('./static/css/'))
        .pipe(livereload());
});

gulp.task('js', function () {
    return gulp.src('./assets/js/*.js')
        .pipe(concat('script.min.js'))
        .pipe(uglify())
        .pipe(gulp.dest('/static/js/'))
        .pipe(livereload())
});

gulp.task('watch', function (){
    livereload.listen();
    gulp.watch('./assets/scss/*.scss', ['scss']);
    gulp.watch('./assets/js/*.js', ['js']);
});

gulp.task('default', ['scss', 'js', 'watch']);
